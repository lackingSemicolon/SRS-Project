Description:
This project will be similar to redbox with a combination of imdb. It'll display a list of movies where users can click on a movie to see a description
and a trailer of the movie.

Team Member:
Tam Nguyen
Kyle Voos
Andres Gonzales-Flores

Status: finished


Requirements:
Meteor:
https://www.meteor.com/

To install, navigate to the repo's 'app' folder and run 'meteor npm install'
	and it should install all the required packages
Then enter 'npm run start'
